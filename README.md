# Student Success Analytics & Placement Prediction System
The Student Success Analytics and Placement Prediction System is a machine learning-based web app that predicts student placement chances using features like CGPA, skills, and internships. It classifies students into tiers and provides company recommendations and skill improvement suggestions

## Features
- Placement prediction using Random Forest
- Tier classification (Tier 1, 2, 3, Not Placed)
- Company recommendations
- Skill improvement suggestions
- Practice problems with links

## Technologies
- Python
- Streamlit
- Scikit-learn
- Pandas

## Run the Project
pip install -r requirements.txt
streamlit run app.py
